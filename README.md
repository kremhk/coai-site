# CoAI site

## Работа с проектом
- Публичный сайт развёрнут на GitHub Pages: <https://kremhk.github.io/coai-site/>.
- Основные файлы: `index.html`, `styles.css`, `script.js`, а также `admin.html` для просмотра заявок.
- При необходимости дорабатывайте контентные блоки напрямую в HTML/CSS — сборки не требуются.

## Админ-панель
- Страница `admin.html` отображает локальные очереди заявок из формы, CRM и почтового канала.
- Доступ выдаётся команде внутри CoAI — данные для входа держим в приватном чек-листе, а не на публичных страницах.
- Если браузер ограничивает `localStorage`, скрипт автоматически переводит панель и форму на хранение в cookies и показывает предупреждение.
